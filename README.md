# Laravel Vue CRUD

CRUD with Laravel API Resource and Vue.

Filters, sorting, pagination and num rows control. Modals and tooltips. 

## Stack

- Laravel Framework 9.x
- Vue 3.x
- Bootstrap 5.x
- Bootstrap Icons
- BootstrapVue3

## Packages

- laravel-mix
- vue3-popper

## Laravel App

- Laravel API Resources
- Repositories
- Form Requests Validation
- Migrations + Seeders
- Models: SoftDeletes, Factory
